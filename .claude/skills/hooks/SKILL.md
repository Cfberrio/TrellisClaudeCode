## Use when
- "Write me hooks."
- "Why are my hooks not working?"
- "What hook formats are there?"
- "Show me high-performing hook examples."
- Any opener / first-3-5-seconds / first-sentence question for organic content, video, posts.

## Output expectations
- For a hook brief: pick the 1-2 best formats from the 8-palette for the user's specific context, then draft 3-5 hooks following the two-part anatomy.
- For a "why aren't my hooks working" diagnosis: name the broken layer (anatomy / format choice / production cadence / specificity), prescribe the fix.
- For swipe-file requests: pull from the library, don't paraphrase Hormozi's exact lines without citation.
- End with the 70-20-10 production assignment for the next batch (70% proven, 20% iteration, 10% experimental).
